
# Personal Portfolio - Mamun Bepari

Welcome to my personal portfolio website. This project showcases my journey as a self-taught **Junior Backend Developer**, my technical skills, and the projects I have built.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS, Django, AOS (Animate On Scroll)
- **Backend Knowledge:** Python, Django, Django REST Framework (DRF)
- **Database:** PostgreSQL, Neon
- **DevOps & Tools:** Docker, Redis, Postman, Git & GitHub

---

## 📂 Folder Structure
```text
PORTFOLIO/
├── images/             # Profile and project screenshots
├── node_modules/       # Tailwind dependencies
├── index.html          # Main HTML structure
├── input.css           # Tailwind source CSS
├── output.css          # Compiled CSS for production
├── tailwind.config.js  # Tailwind configuration
└── README.md           # Project documentation
```

---

## 🌟 Key Features
- Responsive Design: Optimized for mobile, tablet, and desktop.
- Dark/Light Mode: Seamless theme switching with a single click.
- Animations: Smooth scroll animations using AOS library.
- Direct Contact: Integrated links for WhatsApp, LinkedIn, and Email.
